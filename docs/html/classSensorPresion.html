Detailed Description

Clase especializada para dispositivos barometricos.

Clase concreta SensorPresion Representa un sensor que maneja lecturas de tipo int (presión)

Hereda de SensorBase e implementa la funcionalidad especifica para sensores de presion. Almacena mediciones de tipo entero y calcula la media aritmetica de los valores registrados.
