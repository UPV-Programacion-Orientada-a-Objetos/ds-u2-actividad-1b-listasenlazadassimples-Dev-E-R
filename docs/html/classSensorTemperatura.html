Detailed Description

Clase especializada para dispositivos termicos.

Clase concreta SensorTemperatura Representa un sensor que maneja lecturas de tipo float (temperatura)

Hereda de SensorBase e implementa la funcionalidad especifica para sensores de temperatura. Almacena mediciones de tipo float y calcula el valor minimo registrado.
