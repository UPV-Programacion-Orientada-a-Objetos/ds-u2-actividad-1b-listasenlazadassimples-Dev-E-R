Detailed Description

template<typename T>
class ListaSensor< T >

Estructura de datos enlazada generica para almacenamiento dinamico.

Clase de Lista Enlazada Simple Genérica Implementa operaciones básicas de inserción, búsqueda y liberación Cumple con la Regla de los Tres/Cinco para gestión de memoria

Implementa una lista enlazada simple que puede almacenar cualquier tipo de dato. Proporciona operaciones fundamentales de insercion, busqueda, iteracion y liberacion. Cumple con la Regla de los Tres para gestion correcta de memoria dinamica.
